exam.csv 데이터 소개

5개 반, 25명의 학생 성적 데이터입니다. 각 열의 의미는 다음과 같습니다.

  • no : 학생 번호  |  nclass : 반 번호 (1~5)
  • hours : 공부 시간  |  extra : 과외 횟수
  • math : 수학 점수  |  eng : 영어 점수  |  science : 과학 점수
  • tot : 합계 점수  |  test : 합격 여부 (Pass/Fail)

3

원하는 데이터 추출하기 (필터링)

조건을 걸어 특정 행이나 열만 뽑아내는 방법을 배웁니다
⏱ ~15분
3-1. 단일 조건 필터링

대괄호 안에 조건식을 넣어 해당 행만 추출합니다. 조건식은 True/False로 평가됩니다.

Python 
# 1반 학생만
class1 = df[df['nclass'] == 1]
print(class1)

# 수학 점수 70점 이상
math_high = df[df['math'] >= 70]
print(math_high[['no', 'nclass', 'math', 'test']])

# 합격한 학생만
pass_df = df[df['test'] == 'Pass']
print(f"합격 학생 수: {len(pass_df)}명")
3-2. 복합 조건 필터링

&(AND), |(OR) 연산자로 여러 조건을 결합합니다. 각 조건은 반드시 ()로 감싸야 합니다.

Python 
# AND: 수학 70점 이상 AND 영어 75점 이상
top = df[(df['math'] >= 70) & (df['eng'] >= 75)]
print("수학 70↑ & 영어 75↑:", len(top), "명")
print(top[['no', 'nclass', 'math', 'eng', 'test']])

# OR: 수학 80점 이상 OR 과학 80점 이상
excel = df[(df['math'] >= 80) | (df['science'] >= 80)]
print("수학 80↑ OR 과학 80↑:", len(excel), "명")

# isin(): 특정 반만 선택 (2반 또는 4반)
class24 = df[df['nclass'].isin([2, 4])]
print(class24[['no', 'nclass', 'math']])
3-3. query() 메서드 활용

query()는 문자열로 조건을 작성해 더 읽기 쉬운 코드를 만들 수 있습니다.

Python 
# query()로 동일한 조건 - 더 읽기 쉬움
result = df.query("math >= 70 and eng >= 75")
print(result[['no', 'nclass', 'math', 'eng']])

# 합격하고 공부 시간 30시간 이상
hardworking_pass = df.query("test == 'Pass' and hours >= 30")
print(hardworking_pass[['no', 'hours', 'tot', 'test']])
⚠️
주의: 조건 연산자 차이

판다스에서는 and 대신 &, or 대신 |를 사용합니다. query() 안에서만 and/or를 사용할 수 있습니다.

4

그룹별 분석 (groupby)

반별, 합격 여부별로 묶어서 통계를 비교합니다
⏱ ~15분
4-1. 반별 평균 점수

groupby()는 특정 열의 값에 따라 데이터를 묶은 뒤 집계 함수를 적용합니다. Excel의 피벗 테이블과 유사합니다.

Python 
# 반별 수학 평균
print(df.groupby('nclass')['math'].mean().round(1))

# 반별 과목 평균 전체
class_mean = df.groupby('nclass')[['math', 'eng', 'science', 'tot']].mean().round(1)
print(class_mean)

4-2. 반별 합격/불합격 인원
Python 
# 반별 합격/불합격 인원 (crosstab)
ct = pd.crosstab(df['nclass'], df['test'])
print(ct)

# 반별 합격률 (비율로)
ct_pct = pd.crosstab(df['nclass'], df['test'], normalize='index').round(2)
print(ct_pct)
▶ 출력 결과

4-3. 반별 여러 통계 한 번에 (agg)

agg()로 여러 집계 함수를 동시에 적용할 수 있습니다.

Python 
# 반별 합계(tot) 통계: 평균, 최대, 최소, 중앙값
result = df.groupby('nclass')['tot'].agg(['mean', 'max', 'min', 'median']).round(1)
result.columns = ['평균', '최대', '최소', '중앙값']
print(result)
▶ 출력 결과

🔑 핵심 정리

5반이 평균 합계 221.2점으로 1위, 3반이 159.0점으로 가장 낮습니다. groupby('열이름')['집계열'].집계함수() 패턴을 기억하세요!

5

파생변수 만들기

기존 열을 조합해 새로운 분석용 열을 추가합니다
⏱ ~10분
5-1. 과목 평균 점수 열 추가
Python 
# 과목 평균 점수 열 추가
df['avg'] = (df['math'] + df['eng'] + df['science']) / 3
df['avg'] = df['avg'].round(1)
print(df[['no', 'math', 'eng', 'science', 'avg']].head())
5-2. 성적 등급 열 추가 (np.where)

np.where(조건, 참일 때 값, 거짓일 때 값)으로 조건에 따라 값을 다르게 부여할 수 있습니다. 중첩해서 다중 조건도 처리합니다.

Python 
# 합계 기준 등급 부여: A(220↑), B(180↑), C(그 외)
df['grade'] = np.where(df['tot'] >= 220, 'A',
               np.where(df['tot'] >= 180, 'B', 'C'))

print(df[['no', 'nclass', 'tot', 'grade']])
print("\n등급별 인원:")
print(df['grade'].value_counts().sort_index())
▶ 출력 결과
등급별 인원: grade A 7 B 7 C 11 Name: count, dtype: int64
5-3. 공부 효율 지수 추가
Python 
# 공부 효율 = 합계 / 공부시간 (1시간당 점수)
df['efficiency'] = (df['tot'] / df['hours']).round(1)

# 효율이 높은 상위 5명
top5 = df.nlargest(5, 'efficiency')
print(top5[['no', 'nclass', 'hours', 'tot', 'efficiency', 'grade']])

Q1
df.info()를 실행했을 때 확인할 수 없는 정보는 무엇인가요?
  • ① 각 열의 데이터 타입 (int64, object 등)
  • ② 결측치(NaN)가 없는 값의 개수
  • ③ 각 열의 평균값과 최댓값
  • ④ 전체 행(row)의 개수
💡 df.info()는 열 이름·타입·결측치 수를 보여줍니다. 평균·최대값은 df.describe()로 확인합니다.
Q2
수학 점수(math)가 70점 이상이고 합격(test == 'Pass')인 학생만 추출하는 올바른 코드는?
  • ① df[df['math'] >= 70 and df['test'] == 'Pass']
  • ② df[(df['math'] >= 70) & (df['test'] == 'Pass')]
  • ③ df[df['math'] >= 70 | df['test'] == 'Pass']
  • ④ df.filter(math=70, test='Pass')
💡 판다스에서 AND 조건은 &, OR 조건은 |를 사용하며 각 조건은 반드시 괄호 ()로 감쌉니다. Python의 and/or는 사용 불가!
Q3
반(nclass)별 수학(math) 점수 평균을 구하는 올바른 코드는?
  • ① df.mean(['nclass', 'math'])
  • ② df.groupby('math')['nclass'].mean()
  • ③ df.groupby('nclass')['math'].mean()
  • ④ df['nclass']['math'].groupby().mean()
💡 패턴: df.groupby('기준열')['집계열'].집계함수() — 기준이 되는 열을 groupby() 안에, 계산할 열을 [] 안에 씁니다.
Q4
아래 코드를 실행했을 때, 합계(tot) 점수가 195점인 학생의 등급(grade)은?

df['grade'] = np.where(df['tot'] >= 220, 'A',
               np.where(df['tot'] >= 180, 'B', 'C'))
  • ① A등급 (220점 이상)
  • ② B등급 (180점 이상 220점 미만)
  • ③ C등급 (180점 미만)
  • ④ 오류 발생 (조건이 겹침)
💡 195점은 220 미만 → A 탈락, 180 이상 → B 해당. np.where()는 조건을 위에서 아래로 순서대로 체크합니다.
Q5
공부 시간(hours)과 합계 점수(tot)의 관계를 시각화하고, 합격/불합격을 색으로 구분하려 합니다. 가장 적합한 그래프 종류는?
  • ① 히스토그램 — 하나의 변수 분포를 볼 때
  • ② 파이 차트 — 전체 대비 비율을 볼 때
  • ③ 막대 그래프 — 범주별 값을 비교할 때
  • ④ 산점도(scatter) — 두 수치 변수 간 관계를 볼 때
💡 산점도는 두 수치 변수를 x·y 축으로 놓고 점으로 표현해 상관관계 파악에 최적. groupby('test')로 합격 여부를 나눠 색을 다르게 지정합니다.
