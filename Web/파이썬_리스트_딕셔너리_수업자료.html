1. 리스트 — 여러 값을 한 줄에 담기
대괄호 [ ]로 만들고 인덱스로 값을 꺼낸다
📦

리스트 = 칸이 여러 개인 사물함입니다. 각 칸에는 0번부터 번호(인덱스)가 붙어 있어요. 미니 프로젝트에서는 학생 명단을 리스트로 다룹니다.

1만들기 · 값 꺼내기 · 기본 함수
list_basic.py 
# 값이 들어있는 리스트
fruits = ["사과", "바나나", "포도", "수박"]
scores = [85, 92, 78, 95, 88]

# 값 꺼내기 — 인덱스는 0부터 시작!
print(fruits[0])     # 사과
print(fruits[-1])    # 수박 (뒤에서 첫 번째)

# 자주 쓰는 함수들 — 미니 프로젝트의 평균 계산에 사용
print(len(scores))    # 5  (개수)
print(sum(scores))    # 438
print(max(scores))    # 95
print(min(scores))    # 78

# 평균 = 합계 / 개수
average = sum(scores) / len(scores)
print(average)       # 87.6
📋 인덱스 시각화
[0]사과
[1]바나나
[2]포도
[3]수박

방법기준반환값언제 쓰나?
list.pop(i)인덱스삭제된 값꺼내서 다시 쓸 때
list.remove(값)값 자체없음위치 모를 때

2. 슬라이싱 — 원하는 구간만 잘라내기
[시작 : 끝 : 간격] 문법으로 부분 리스트 만들기
🍞

슬라이싱 = 식빵 자르기입니다. [2:5]는 "2번 칼선부터 5번 칼선까지" 자르는 것. 끝 번호는 포함되지 않습니다(2, 3, 4까지만).

slice.py 
scores = [85, 92, 78, 95, 88, 76, 91, 83]
#          0   1   2   3   4   5   6   7   ← 인덱스

# [시작 : 끝]  — 끝은 포함 안 됨!
print(scores[2:5])    # [78, 95, 88]  (2,3,4번)

# 시작 생략 → 0부터 / 끝 생략 → 끝까지
print(scores[:3])     # [85, 92, 78]  (앞에서 3개)
print(scores[5:])     # [76, 91, 83]  (5번부터 끝까지)

# 음수 인덱스 — 뒤에서부터
print(scores[-3:])    # [76, 91, 83]  (마지막 3개)
print(scores[:-2])    # 마지막 2개 빼고 다

# [시작 : 끝 : 간격]
print(scores[::2])    # [85, 78, 88, 91]  (2칸씩)

# 거꾸로 뒤집기 — 음수 간격
print(scores[::-1])   # [83, 91, 76, 88, 95, 78, 92, 85]

# 전체 복사 — 원본과 분리된 새 리스트
copied = scores[:]
copied[0] = 0
print(scores[0])     # 85  (원본은 그대로!)
📐 슬라이싱 시각 가이드
scores = [85, 92, 78, 95, 88, 76, 91, 83]
인덱스: [ 0, 1, 2, 3, 4, 5, 6, 7]
[2:5] → 2,3,4번 (5번 미포함)
[::-1] → 거꾸로 뒤집기
[:3] → 앞에서 3개만

🎯 미니 프로젝트와의 연결

미니 프로젝트에서 한 학생의 점수 리스트 scores = [85, 92, 78]가 있을 때, scores[:2]로 처음 두 과목만 잘라 그 평균을 구하거나, scores[-1]로 마지막 과목 점수만 꺼낼 수 있습니다.

3. 딕셔너리 — 이름표(key)가 붙은 값(value)
중괄호 { }와 콜론 :으로 만드는 "사전" 형태
📖

딕셔너리 = 진짜 사전입니다. 리스트는 0, 1, 2 같은 번호로 값을 꺼냈지만, 딕셔너리는 이름(키)으로 꺼냅니다. 미니 프로젝트에서는 학생 한 명의 이름·학년·점수를 딕셔너리로 묶습니다.

1만들기 · 값 꺼내기
dict_basic.py 
# 학생 한 명을 딕셔너리로 표현 — 미니 프로젝트의 기본 구조!
student = {
    "name": "민준",
    "grade": 1,
    "scores": [85, 92, 78]
}

# 값 꺼내기 — [키] 사용
print(student["name"])       # 민준
print(student["scores"])     # [85, 92, 78]
print(student["scores"][0])  # 85 (리스트의 첫 값)

# 키 존재 여부 확인 — in 연산자
print("name" in student)    # True
print("gender" in student)  # False

# 안전하게 꺼내기 — get() (없는 키여도 에러 X)
print(student.get("gender", "미입력"))  # 미입력
⚠️
없는 키 → KeyError student["gender"]처럼 없는 키를 대괄호로 꺼내면 에러가 납니다. 확실하지 않다면 get() 또는 in으로 먼저 확인하세요.
2추가 · 수정 · 삭제
dict_crud.py 
student = {"name": "민준", "grade": 1}

# 추가 — 새 키에 값 대입
student["scores"] = [85, 92, 78]
print(student)
# {'name': '민준', 'grade': 1, 'scores': [85, 92, 78]}

# 수정 — 같은 문법, 이미 있는 키면 덮어쓰기
student["grade"] = 2

# 삭제 — del 또는 pop
del student["grade"]
print(student)         # {'name': '민준', 'scores': [...]}

# pop — 꺼내면서 삭제 (반환값 있음)
removed = student.pop("scores")
print(removed)         # [85, 92, 78]
💡
같은 문법, 다른 동작 student["키"] = 값은 키가 없으면 추가, 있으면 수정이 됩니다.

4. 종합 — for 반복 · 함수로 묶기
리스트·딕셔너리를 for로 순회하고 함수로 재사용한다
for리스트 순회 + if로 골라내기
for_list.py 
scores = [85, 92, 78, 95, 88]

# 모든 점수 출력
for s in scores:
    print(s)

# 합계와 평균 구하기 — 미니 프로젝트의 핵심 패턴
total = 0
for s in scores:
    total += s
average = total / len(scores)
print(f"합계 {total}, 평균 {average:.1f}")

# 조건에 맞는 값만 골라 새 리스트로 모으기
high = []
for s in scores:
    if s >= 85:
        high.append(s)
print(high)         # [85, 92, 95, 88]
for딕셔너리 순회 — items()
for_dict.py 
score_table = {
    "민준": 85,
    "서연": 92,
    "지호": 78,
}

# 키와 값을 함께 — items()
for name, score in score_table.items():
    print(f"{name}: {score}점")

# 조건에 맞는 항목만 추리기
passers = []
for name, score in score_table.items():
    if score >= 80:
        passers.append(name)
print(passers)      # ['민준', '서연']
def함수로 묶기 — 같은 작업을 이름 하나로 재사용
🛠️

함수 = 자판기입니다. 동전(인자)을 넣으면 음료(반환값)가 나오죠. 미니 프로젝트의 모든 기능은 함수로 만듭니다.

func.py 
# 점수 리스트를 받아 평균을 돌려주는 함수
def get_average(score_list):
    if len(score_list) == 0:
        return 0
    return sum(score_list) / len(score_list)

print(get_average([85, 92, 78]))   # 85.0
print(get_average([]))               # 0


# 평균 점수를 받아 등급을 돌려주는 함수
def get_grade(avg):
    if   avg >= 90: return "A"
    elif avg >= 80: return "B"
    elif avg >= 70: return "C"
    else:             return "F"

print(get_grade(85))   # B
print(get_grade(95))   # A

🎯 이 두 함수가 곧 미니 프로젝트의 부품

다음 섹션의 학생 점수 관리 시스템은 get_average와 get_grade를 그대로 가져다 쓰면서, 학생 추가·삭제·전체 출력 함수를 위에 얹는 구조입니다.

5. 미니 프로젝트 — 학생 점수 관리 시스템
지금까지 배운 것을 모두 모아 작은 프로그램을 만든다

학생들의 정보를 딕셔너리의 리스트로 저장하고, 추가·삭제·검색·통계 기능을 함수로 만듭니다. 이 패턴은 실제 데이터 분석에서 CSV 한 줄 한 줄을 다룰 때와 똑같은 구조입니다.

🧩
여기까지의 부품 점검 ① 리스트 append·pop → 학생 추가·삭제  |  ② 슬라이싱 → 일부 학생/점수만 보기  |  ③ 딕셔너리 [키] → 학생 한 명의 정보 꺼내기  |  ④ for+if → 명단 순회·합격자 찾기  |  ⑤ def → 기능을 함수로 묶기
project_grades.py 
# 학생 정보 = 리스트(명단) + 딕셔너리(학생 한 명)
students = [
    {"name": "민준", "grade": 1, "scores": [85, 92, 78]},
    {"name": "서연", "grade": 2, "scores": [95, 88, 90]},
    {"name": "지호", "grade": 1, "scores": [72, 68, 75]},
]


# ─── 함수 1: 학생 추가 ────────────────────
def add_student(student_list, name, grade, scores):
    new_student = {"name": name, "grade": grade, "scores": scores}
    student_list.append(new_student)
    print(f"✅ {name} 학생을 추가했습니다.")


# ─── 함수 2: 학생 삭제 (이름으로 찾기) ─────
def remove_student(student_list, name):
    for i, s in enumerate(student_list):
        if s["name"] == name:
            student_list.pop(i)
            print(f"🗑️ {name} 학생을 삭제했습니다.")
            return
    print(f"⚠️ {name} 학생을 찾을 수 없습니다.")


# ─── 함수 3: 학생 찾기 ────────────────────
def find_student(student_list, name):
    for s in student_list:
        if s["name"] == name:
            return s
    return None


# ─── 함수 4: 평균 + 등급 (앞 섹션 부품 재사용) ─
def get_average(score_list):
    return sum(score_list) / len(score_list)

def get_grade(avg):
    if   avg >= 90: return "A"
    elif avg >= 80: return "B"
    elif avg >= 70: return "C"
    else:             return "F"


# ─── 함수 5: 전체 명단 출력 ────────────────
def print_all(student_list):
    print("\n📋 학생 명단")
    print("─" * 42)
    for s in student_list:
        avg = get_average(s["scores"])
        print(f"{s['name']:<6} | {s['grade']}학년 | "
              f"평균 {avg:5.1f} | 등급 {get_grade(avg)}")


# ─── 함수 6: 슬라이싱 활용 — 상위 N명 미리보기 ─
def preview(student_list, n=2):
    print(f"\n👀 처음 {n}명 미리보기")
    for s in student_list[:n]:        # ← 슬라이싱!
        print(f"  - {s['name']} ({s['grade']}학년)")


# ─── 실행 ────────────────────────────────
print_all(students)

add_student(students, "예린", 3, [88, 91, 85])
remove_student(students, "지호")

print_all(students)
preview(students, 2)

found = find_student(students, "서연")
print(f"\n🔍 서연 정보: {found}")
실행 결과 📋 학생 명단 ────────────────────────────────────────── 민준 | 1학년 | 평균 85.0 | 등급 B 서연 | 2학년 | 평균 91.0 | 등급 A 지호 | 1학년 | 평균 71.7 | 등급 C ✅ 예린 학생을 추가했습니다. 🗑️ 지호 학생을 삭제했습니다. 📋 학생 명단 ────────────────────────────────────────── 민준 | 1학년 | 평균 85.0 | 등급 B 서연 | 2학년 | 평균 91.0 | 등급 A 예린 | 3학년 | 평균 88.0 | 등급 B 👀 처음 2명 미리보기 - 민준 (1학년) - 서연 (2학년) 🔍 서연 정보: {'name': '서연', 'grade': 2, 'scores': [95, 88, 90]}

def update_scores(student_list, name, new_scores):
    for s in student_list:
        if s["name"] == name:
            s["scores"] = new_scores
            return
    print(f"⚠️ {name} 학생 없음")

def top_student(student_list):
    top_name = ""
    top_avg = -1
    for s in student_list:
        avg = get_average(s["scores"])
        if avg > top_avg:
            top_avg = avg
            top_name = s["name"]
    return top_name
