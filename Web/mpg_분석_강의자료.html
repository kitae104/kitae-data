📌
mpg.csv 데이터 소개

1999~2008년 사이 출시된 자동차 234대의 연비 데이터입니다. 각 열의 의미는 다음과 같습니다.

  • manufacturer: 제조사  |  model: 모델명  |  year: 출시 연도
  • displ: 배기량(L)  |  cyl: 실린더 수  |  trans: 변속기 종류
  • drv: 구동 방식 (f=전륜, r=후륜, 4=4륜)
  • cty: 도시 연비(mpg)  |  hwy: 고속도로 연비(mpg)
  • fl: 연료 종류  |  category: 자동차 종류

파생변수 만들기 (연비 평균 & 등급)

4-1. 통합 연비 평균(avg) 열 추가

# 도시+고속도로 연비 평균
df['avg'] = (df['cty'] + df['hwy']) / 2

print(df[['manufacturer', 'model', 'cty', 'hwy', 'avg']].head())
print("\n전체 평균 연비:", df['avg'].mean().round(1))

4-2. 연비 합격 판정 (np.where)

전체 평균(약 20mpg)을 기준으로 합격/불합격을 나눕니다. np.where(조건, 참값, 거짓값) 패턴입니다.

Python 
# 평균 연비 기준으로 합격/불합격 판정
df['test'] = np.where(df['avg'] >= 20, 'pass', 'fail')
print(df['test'].value_counts())
▶ 출력 결과
test pass 128 fail 106 Name: count, dtype: int64
4-3. 연비 등급(A/B/C) 부여

np.where()를 중첩해서 세 등급으로 나눕니다. 평균 연비 30 이상 → A, 20 이상 → B, 미만 → C.

Python 
# 연비 등급: A(avg>=30), B(avg>=20), C(그 외)
df['grade'] = np.where(df['avg'] >= 30, 'A',
               np.where(df['avg'] >= 20, 'B', 'C'))

print("등급별 대수:")
print(df['grade'].value_counts().sort_index())

# 등급별 카테고리 분포 (crosstab)
ct = pd.crosstab(df['grade'], df['category'])
print(ct)
▶ 출력 결과
등급별 대수: grade A 10 B 118 C 106 Name: count, dtype: int64 category 2seater compact midsize minivan pickup subcompact suv grade A 0 8 0 0 0 2 0 B 5 39 41 11 0 21 1 C 0 0 0 0 33 12 61
💡
분석 결과 해석

A등급(최고 연비)은 compact와 subcompact에 집중되어 있고, C등급(낮은 연비)은 pickup과 suv가 대부분을 차지합니다. 차체 크기와 연비는 반비례 관계임을 확인할 수 있습니다.

변수 간 관계 분석

5-1. 상관관계 분석 (corr)

corr()은 두 변수가 얼마나 함께 움직이는지를 -1 ~ +1 사이 값으로 나타냅니다. 1에 가까울수록 양의 상관, -1에 가까울수록 음의 상관.

Python 
# 수치형 변수 간 상관관계
corr_cols = ['displ', 'cyl', 'cty', 'hwy', 'avg']
corr_df = df[corr_cols].corr().round(2)
print(corr_df)

displ_avg = df.groupby('displ_group', observed=True)['avg'].mean().round(1)

print(displ_avg)

🔑 분석 결론

배기량(displ)과 연비(avg)의 상관계수는 -0.79로 강한 음의 상관관계. 즉, 배기량이 클수록 연비가 나쁩니다. 소형(1~2L) 평균 26.6mpg vs 초대형(4L↑) 16.0mpg로 약 67% 차이.

6-3. 카테고리별 연비 박스플롯

박스플롯(상자 수염 그림)은 데이터의 중앙값, 사분위 범위, 이상치를 동시에 보여주는 강력한 그래프입니다.

Python 
# 카테고리별 고속도로 연비 데이터 준비
cat_order = df.groupby('category')['hwy'].median().sort_values(ascending=False).index
data_by_cat = [df[df['category'] == c]['hwy'].values for c in cat_order]

fig, ax = plt.subplots(figsize=(11, 6))
bp = ax.boxplot(data_by_cat, labels=cat_order,
                patch_artist=True, notch=False)

# 박스 색상 지정
box_colors = ['#bfdbfe', '#bbf7d0', '#fde68a', '#ddd6fe',
              '#fecaca', '#a5f3fc', '#e5e7eb']
for patch, color in zip(bp['boxes'], box_colors):
    patch.set_facecolor(color)

ax.set_title('자동차 종류별 고속도로 연비 분포', fontsize=13, fontweight='bold')
ax.set_xlabel('자동차 종류')
ax.set_ylabel('고속도로 연비 (mpg)')
ax.grid(True, axis='y', alpha=0.3)
plt.tight_layout()
plt.show()

📝 5문제
Q1
mpg 데이터에서 제조사(manufacturer)가 몇 종류인지 확인하는 올바른 코드는?
  • ① df['manufacturer'].count() — 전체 행 수를 셈
  • ② df['manufacturer'].nunique() — 고유값 개수를 셈
  • ③ len(df['manufacturer']) — 전체 행 수를 셈
  • ④ df.shape[1] — 열(column) 수를 셈
💡 count()는 결측치를 제외한 전체 행 수, nunique()는 중복 없는 고유값 개수입니다. 목록 자체가 필요하면 unique()를 사용합니다.
Q2
고속도로 연비(hwy)가 높은 자동차 상위 5대를 출력하는 코드로 올바른 것은?
  • ① df.sort_values('hwy').head(5)
  • ② df.nlargest(5, 'hwy')
  • ③ df.head(5)['hwy']
  • ④ df['hwy'].max(5)
💡 ①번은 오름차순 정렬 후 앞 5개(가장 낮은 값)를 가져옵니다. 내림차순 상위를 원하면 sort_values('hwy', ascending=False).head(5) 또는 nlargest(5, 'hwy')가 더 간결합니다.
Q3
아래 코드를 실행한 뒤, 평균 연비(avg)가 17.5인 자동차의 test 값은?

df['avg'] = (df['cty'] + df['hwy']) / 2
df['test'] = np.where(df['avg'] >= 20, 'pass', 'fail')
  • ① 'fail' — 17.5 < 20이므로
  • ② 'pass' — 조건을 만족하므로
  • ③ NaN — avg 값이 정수가 아니므로
  • ④ 오류 발생 — avg 열이 없으므로
💡 17.5는 20보다 작으므로 조건 df['avg'] >= 20이 False. np.where(조건, 참값, 거짓값)에서 거짓값인 'fail'이 할당됩니다.
Q4
배기량(displ)과 연비(avg)의 상관계수가 -0.79일 때, 이를 올바르게 해석한 것은?
  • ① 배기량이 클수록 연비가 높아지는 강한 경향이 있다
  • ② 배기량과 연비는 거의 관계가 없다 (0에 가까우므로)
  • ③ 배기량이 클수록 연비가 낮아지는 강한 경향이 있다
  • ④ 배기량과 연비는 항상 정확히 반비례한다
💡 상관계수 범위: -1 ~ +1. 음수(-) = 한 쪽이 증가하면 다른 쪽 감소, |값|이 클수록 관계 강함. -0.79는 강한 음의 상관. -1.0일 때만 "항상 정확히 반비례"입니다.
Q5
아래 코드에서 pd.cut()의 역할로 가장 올바른 설명은?

df['displ_group'] = pd.cut(df['displ'],
  bins=[1, 2, 3, 4, 8],
  labels=['소형','중형','대형','초대형'])
  • ① 배기량이 1~8 사이인 행만 필터링한다
  • ② 배기량 데이터를 오름차순으로 정렬한다
  • ③ 연속형 수치를 구간별로 나눠 범주 레이블을 붙인다
  • ④ 배기량 평균을 기준으로 상위/하위 두 그룹으로 나눈다
💡 pd.cut()은 연속형 숫자를 bins(구간 경계)로 잘라 labels(이름)를 부여합니다. 나이대(10대/20대…), 점수 등급 등 구간화에 활용합니다. 균등 분할이 필요하면 pd.qcut()을 사용합니다.
