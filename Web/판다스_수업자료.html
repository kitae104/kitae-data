Series - 한 열의 데이터
인덱스 + 값의 쌍으로 이루어진 1차원 데이터 구조

📌 Series란?

Series = 엑셀의 '한 열' 입니다. 값들의 목록과 각 값에 붙는 번호(인덱스)로 구성됩니다. 번호 대신 이름을 인덱스로 사용할 수도 있습니다.

02_series.py 
# 학생들의 수학 점수로 Series를 만들어봅니다
math_scores = pd.Series([85, 92, 78, 95, 88])

print("=== 수학 점수 Series ===")
print(math_scores)

# 인덱스를 이름으로 지정할 수도 있습니다
math_scores_named = pd.Series(
    [85, 92, 78, 95, 88],
    index=['김민준', '이서연', '박지호', '최예린', '정도윤']
)

# 특정 학생 점수 조회
print(f"이서연 점수: {math_scores_named['이서연']}점")

# 기본 통계 계산
print(f"평균: {math_scores_named.mean():.1f}점")
print(f"최고: {math_scores_named.max()}점")

데이터 선택하기 (loc & iloc)
원하는 행/열만 골라내는 방법
🔍

loc = 이름으로 찾기 (「김민준 어디있어?」)
iloc = 번호로 찾기 (「3번 자리 학생!」) — i는 integer(정수)의 약자

05_select.py 
## 열(컬럼) 선택 ##
# 열 하나 선택 → Series 반환
df['수학']

# 여러 열 선택 → DataFrame 반환 (대괄호 두 겹!)
df[['이름', '수학', '영어']]

## 행 선택 ##
# iloc - 위치(번호)로 선택 (0부터 시작!)
df.iloc[0]          # 첫 번째 행
df.iloc[1:4]       # 1~3번 행 (4번 미포함)
df.iloc[0:3, 0:3]  # 0~2행, 0~2열

# loc - 인덱스 이름으로 선택
df.loc[0]              # 인덱스가 0인 행
df.loc[0:2, ['이름', '수학']]  # loc은 끝 번호 포함!
ℹ️

iloc vs loc의 슬라이싱 차이:
iloc[1:4] → 1, 2, 3번 (4번 미포함, 파이썬 일반 규칙)
loc[1:3] → 1, 2, 3번 (3번 포함! loc은 양쪽 다 포함)

결측값 확인 및 처리
NaN(Not a Number) - 값이 없는 빈 칸 다루기
01_missing_values.py 
import pandas as pd


# 결측값이 포함된 데이터 만들기 (np.nan = 값 없음)
data = {
    '이름':  ['김민준', '이서연', '박지호', '최예린', '정도윤'],
    '국어':  [85, 92, 78, None, 88],   # 최예린 국어 결측
    '수학':  [92, 78, None, 88, 72],   # 박지호 수학 결측
    '영어':  [88, None, 82, 91, 85]    # 이서연 영어 결측
}
df = pd.DataFrame(data)

## 결측값 확인 ##
# isnull() → 각 값이 결측인지 True/False
df.isnull()

# 열별 결측값 개수
print(df.isnull().sum())

# 결측값 비율 계산
missing_ratio = (df.isnull().sum() / len(df) * 100).round(1)
print(missing_ratio)

## 결측값 처리 ##

# 방법 1: 결측값 있는 행 삭제
df_dropped = df.dropna()

# 방법 2: 평균으로 채우기
df_filled = df.copy()  # 원본 보존을 위해 복사
for col in ['국어', '수학', '영어']:
    col_mean = df_filled[col].mean()
    df_filled[col] = df_filled[col].fillna(col_mean).round(1)

# 방법 3: 특정 값으로 채우기
df.fillna(0)        # 0으로 채우기
df.fillna('없음')   # 문자로 채우기

산점도는 두 값을 x축·y축에 각각 놓고 점으로 찍어봅니다.
점들이 오른쪽 위 방향으로 모이면 → 두 값이 함께 커지는 관계입니다.

plt.xlabel('수학 점수', fontproperties=fontprop)
plt.ylabel('영어 점수', fontproperties=fontprop)
plt.xticks(fontproperties=fontprop)
plt.show()
ℹ️

점에 이름 표시하기 — plt.annotate() 로 각 점 옆에 이름을 붙일 수 있습니다.

# 수학·영어 점수 변화 비교
plt.plot(이름, 수학, marker='o', label='수학', color='steelblue')
plt.plot(이름, 영어, marker='s', label='영어', color='tomato')

plt.title('학생별 수학 vs 영어 점수', fontproperties=fontprop)
plt.xlabel('이름', fontproperties=fontprop)
plt.ylabel('점수', fontproperties=fontprop)
plt.legend(prop=fontprop)   # 범례 표시
plt.grid(True)
plt.show()

📊 그래프 모양

3
막대 그래프 — plt.bar()
항목별 값의 크기를 비교할 때 사용합니다
💡

막대 그래프는 높이로 값을 비교합니다.
가로 방향은 plt.barh() 를 씁니다 (h = horizontal).

03_막대그래프.py 
# 세로 막대 그래프
plt.bar(이름, 수학, color='steelblue')

plt.title('학생별 수학 점수', fontproperties=fontprop)
plt.xlabel('이름', fontproperties=fontprop)
plt.ylabel('점수', fontproperties=fontprop)
plt.xticks(fontproperties=fontprop)
plt.ylim(0, 100)  # y축 범위
plt.show()
ℹ️

가로 막대로 바꾸기 → plt.bar() 대신 plt.barh() 를 씁니다.
가로 막대는 항목 이름이 길 때 더 보기 좋습니다.

03_막대그래프_가로.py 
# 가로 막대 그래프 (barh)
plt.barh(이름, 수학, color='steelblue')

plt.title('학생별 수학 점수', fontproperties=fontprop)
plt.xlabel('점수', fontproperties=fontprop)
plt.ylabel('이름', fontproperties=fontprop)
plt.yticks(fontproperties=fontprop)
plt.show()

📊 그래프 모양

4
파이 차트 — plt.pie()
전체에서 각 항목이 차지하는 비율을 볼 때 사용합니다
💡

파이 차트는 피자를 나눠 먹듯이 전체 비율을 보여줍니다.
autopct='%1.1f%%' → 각 조각에 퍼센트 숫자를 자동으로 표시합니다.

04_파이차트.py 
# 새 데이터: 반 학생이 좋아하는 계절 투표
계절  = ['봄', '여름', '가을', '겨울']
투표  = [8, 14, 10, 6]

# autopct → 퍼센트 자동 표시
plt.pie(투표, labels=계절, autopct='%1.1f%%',
       textprops={'fontproperties': fontprop})

plt.title('좋아하는 계절 비율', fontproperties=fontprop)
plt.show()
ℹ️

특정 조각 강조하기 → explode 옵션으로 원하는 조각을 살짝 분리합니다.
0이면 제자리, 0.1이면 약간 바깥으로 떨어집니다.

04_파이차트_강조.py 
# explode: 각 조각을 얼마나 분리할지 (0=그대로, 0.1=살짝 분리)
분리 = [0, 0.1, 0, 0]   # 여름만 강조

plt.pie(투표, labels=계절,
       autopct='%1.1f%%',
       explode=분리,
       textprops={'fontproperties': fontprop})

plt.title('좋아하는 계절 비율', fontproperties=fontprop)
plt.show()

📊 그래프 모양

5
박스 플롯 — plt.boxplot()
데이터가 어떻게 퍼져 있는지 한 눈에 볼 때 사용합니다
💡

박스 플롯은 상자 하나로 데이터의 분포를 보여줍니다.
상자 안의 선 = 중앙값, 상자 = 절반의 데이터 범위, 수염 = 전체 범위

📦 박스 플롯 구조 읽는 법

05_박스플롯.py 
# 새 데이터: 국어·수학·영어 점수 목록
국어 = [72, 80, 85, 88, 90, 92, 78, 65]
수학 = [60, 75, 82, 88, 95, 70, 55, 90]
영어 = [78, 82, 79, 85, 88, 91, 76, 84]

# 세 과목을 한 번에 박스 플롯으로
plt.boxplot([국어, 수학, 영어],
            labels=['국어', '수학', '영어'])

plt.title('과목별 점수 분포', fontproperties=fontprop)
plt.ylabel('점수', fontproperties=fontprop)
plt.xticks(fontproperties=fontprop)
plt.show()
ℹ️

상자에 색 넣기 → patch_artist=True 옵션을 추가합니다.

05_박스플롯_색추가.py 
# patch_artist=True → 상자 안을 색으로 채우기
bp = plt.boxplot([국어, 수학, 영어],
                  labels=['국어', '수학', '영어'],
                  patch_artist=True)

# 각 상자에 색 지정
색목록 = ['lightblue', 'lightyellow', 'lightgreen']
for 상자, 색 in zip(bp['boxes'], 색목록):
    상자.set_facecolor(색)

plt.title('과목별 점수 분포', fontproperties=fontprop)
plt.ylabel('점수', fontproperties=fontprop)
plt.xticks(fontproperties=fontprop)
plt.show()

5가지 그래프 핵심 정리
언제 어떤 그래프를 쓰는지 한 눈에 확인하세요
그래프코드언제 쓰나요?
🔵 산점도 plt.scatter(x, y) 두 값의 관계(상관관계) 확인
📈 선 그래프 plt.plot(x, y) 시간에 따른 변화·추이 표현
📊 막대 그래프 plt.bar(x, y) 항목별 크기 비교
🥧 파이 차트 plt.pie(값, labels=항목) 전체에서 각 항목의 비율
📦 박스 플롯 plt.boxplot([데이터...]) 데이터 분포·이상값 확인
공통_패턴.py — 모든 그래프에 공통으로 쓰는 코드 
# 그래프 그리기 (종류에 맞는 명령어 선택)
plt.scatter(x, y)    # 산점도
plt.plot(x, y)       # 선 그래프
plt.bar(x, y)        # 막대 그래프 (세로)
plt.pie(값, labels=항목)  # 파이 차트
plt.boxplot([데이터])  # 박스 플롯

# 공통 꾸미기 옵션
plt.title('제목', fontproperties=fontprop)         # 그래프 제목
plt.xlabel('x축 이름', fontproperties=fontprop)  # x축 이름 (산점도·선·막대)
plt.ylabel('y축 이름', fontproperties=fontprop)  # y축 이름
plt.legend(prop=fontprop)             # 범례 (선이 2개 이상일 때)
plt.grid(True)        # 격자선 표시

# 출력 (마지막에 반드시!)
plt.show()
