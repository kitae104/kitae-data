파일 불러오기

가장 먼저 해야 할 일 — CSV 파일을 판다스로 읽기

판다스에서 CSV 파일을 읽을 때는 pd.read_csv()를 사용합니다.
이 파일은 한글이 포함되어 있어서 encoding='cp949'를 꼭 써줘야 합니다.

💡 비유: read_csv()는 엑셀 파일을 여는 것과 같아요.
파일 이름을 넣으면 표(DataFrame) 형태로 불러옵니다.
step1.py 
import pandas as pd          # 판다스 불러오기

# CSV 파일 읽기
# encoding='cp949' : 한글 파일을 올바르게 읽기 위해 필요합니다
df = pd.read_csv('data1.csv', encoding='cp949')

print("불러오기 완료!")
print(f"행(row) 수 : {df.shape[0]}")    # 몇 줄?
print(f"열(column) 수 : {df.shape[1]}")  # 몇 칸?
출력 결과불러오기 완료! 행(row) 수 : 18 열(column) 수 : 310
💡

18행은 전국 1개 + 17개 시도, 310열은 행정구역 이름 + 연령별 남녀 인구 항목들입니다.

숫자 데이터 정제하기

쉼표가 들어간 숫자 문자열 → 진짜 숫자로 바꾸기

CSV 파일에서 숫자가 "51,248,233"처럼 쉼표가 포함된 문자열로 읽혀집니다.
계산을 하려면 쉼표를 지우고 진짜 숫자로 바꿔야 합니다.

💡 비유: 숫자 "1,000"은 컴퓨터에게는 그냥 글자입니다.
계산하려면 쉼표를 지워서 1000이라는 진짜 숫자로 만들어야 해요.
step3.py — 쉼표 제거 + 숫자 변환 
# 원본은 보존하고, 복사본을 만들어서 작업합니다
df2 = df.copy()

# 첫 번째 열(행정구역)을 제외한 나머지 열 모두 처리
for col in df2.columns[1:]:
    df2[col] = (df2[col]
                .astype(str)            # 일단 문자열로
                .str.replace(',', '')  # 쉼표 제거
                .str.strip())           # 앞뒤 공백 제거
    df2[col] = pd.to_numeric(df2[col], errors='coerce')  # 숫자로 변환

# 지역명도 깔끔하게 정리
# "서울특별시  (1100000000)" → "서울특별시"
df2['지역명'] = df2['행정구역'].str.split('  ').str[0]

print("정제 완료!")
print(df2['지역명'].tolist())
출력 결과정제 완료! ['전국', '서울특별시', '부산광역시', '대구광역시', '인천광역시', '광주광역시', '대전광역시', '울산광역시', '세종특별자치시', '경기도', '강원특별자치도', '충청북도', '충청남도', '전북특별자치도', '전라남도', '경상북도', '경상남도', '제주특별자치도']

step4a.py — 서울 총인구 꺼내기 
# 서울특별시 행만 필터링
seoul = df2[df2['지역명'] == '서울특별시'].iloc[0]

# 총인구, 남성, 여성 출력
print(f"서울 총인구 : {seoul['2024년09월_계_총인구수']:,.0f} 명")
print(f"서울 남성   : {seoul['2024년09월_남_총인구수']:,.0f} 명")
print(f"서울 여성   : {seoul['2024년09월_여_총인구수']:,.0f} 명")
출력 결과서울 총인구 : 9,350,995 명 서울 남성 : 4,515,791 명 서울 여성 : 4,835,204 명
step4b.py — 전체 시도 총인구 표로 보기 
# 전국 행 제외, 시도만 추출
sido = df2[df2['지역명'] != '전국'].copy()

# 필요한 열만 골라서, 인구 많은 순으로 정렬
result = sido[['지역명',
               '2024년09월_계_총인구수',
               '2024년09월_남_총인구수',
               '2024년09월_여_총인구수']]

result = result.sort_values('2024년09월_계_총인구수', ascending=False)
result.columns = ['지역', '총인구', '남성', '여성']  # 열 이름 줄이기

print(result.to_string(index=False))
출력 결과 지역 총인구 남성 여성 경기도 13,680,645 6,877,300 6,803,345 서울특별시 9,350,995 4,515,791 4,835,204 경상남도 3,232,297 1,628,643 1,603,654 부산광역시 3,273,473 1,593,783 1,679,690 인천광역시 3,016,520 1,507,439 1,509,081 ...

핵심 코드 정리

5

그래프로 시각화하기

막대 그래프와 선 그래프로 인구 데이터를 눈에 보이게!

matplotlib 라이브러리를 사용하면 데이터를 그래프로 그릴 수 있습니다.
먼저 한글 폰트 설정을 한 번만 해주고, 그래프를 그려봅니다.

step5a.py — 시도별 총인구 가로 막대 그래프 
import matplotlib.pyplot as plt
import matplotlib

# 한글 폰트 설정 (Windows)
matplotlib.rc('font', family='Malgun Gothic')
# Mac 사용자는: matplotlib.rc('font', family='AppleGothic')
matplotlib.rcParams['axes.unicode_minus'] = False

# 오름차순 정렬 → 가로 막대에서 큰 값이 위에 오게
sido_sorted = result.sort_values('총인구')

fig, ax = plt.subplots(figsize=(10, 7))
ax.barh(sido_sorted['지역'], sido_sorted['총인구'],
       color='#3B82F6', alpha=0.85)

ax.set_title('2024년 9월 시도별 총인구', fontsize=14, fontweight='bold')
ax.set_xlabel('인구수 (명)')
ax.grid(axis='x', alpha=0.3)
plt.tight_layout()
plt.show()
출력 결과 → 경기도가 가장 긴 막대(약 1,368만)로 표시되고, 세종특별자치시가 가장 짧은 막대(약 39만)로 표시됩니다.
step5b.py — 전국 연령별 남녀 인구 선 그래프 
# 전국 행 꺼내기
전국 = df2[df2['지역명'] == '전국'].iloc[0]

# 0세~99세 남성·여성 인구 리스트 만들기
ages = list(range(100))
남성  = [전국[f'2024년09월_남_{i}세'] for i in ages]
여성  = [전국[f'2024년09월_여_{i}세'] for i in ages]

# 선 그래프 그리기
fig, ax = plt.subplots(figsize=(12, 5))
ax.plot(ages, 남성, label='남성', color='#3B82F6', linewidth=2)
ax.plot(ages, 여성, label='여성', color='#EF4444', linewidth=2)

ax.set_title('전국 연령별 남녀 인구 (2024년 9월)', fontsize=14, fontweight='bold')
ax.set_xlabel('나이 (세)')
ax.set_ylabel('인구수 (명)')
ax.legend(fontsize=12)
ax.grid(alpha=0.3)
plt.tight_layout()
plt.show()
출력 결과 → 50~60대 인구가 가장 많고, 0세로 갈수록 적어지는 그래프가 그려집니다. 어린 나이에서는 남성이 약간 많고, 70세 이후부터는 여성이 훨씬 많습니다.
✅

그래프 기본 구조
plt.subplots(figsize=(가로, 세로)) → 도화지 만들기
ax.barh() / ax.bar() / ax.plot() → 그래프 종류 선택
ax.set_title() / set_xlabel() / legend() → 제목·레이블 추가
plt.show() → 화면에 출력
